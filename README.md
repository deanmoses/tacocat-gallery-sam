# tacocat-gallery-sam
Tacocat Gallery system implemented on Amazon AWS Serverless Application Model (SAM)

# Installation
Run `npm install` on each nested SAM app, such as:
```
    cd image_api/image_api_app
    npm install

    cd state_machine/state_machine_app
    npm install
```

# Build
From root:
```
npm run build
```

# Deploy
From root:
```
npm run deploy
```
